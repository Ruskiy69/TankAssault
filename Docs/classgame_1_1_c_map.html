Detailed Description

Throughout Collapse, there are three layers to every map. The lowest layer is the terrain map. It describes terrain tiles, each which have unique properties such as impassibility or slower movement speed. The next highest layer is the collision map. This one describes collision data for all on-screen tiles. Then finally, at the highest level, is the objective map, which specifies spawn points, player objectives, AI points-of-interest, etc.
