bool CPlayer::Init ( game::CSettings &  Settings)
virtual

The main sprite image is loaded here, which resides in "Data/Images" for the time being. Since there are multiple sprites in the sheet, they are each cut out, and the GL_Entity is resized to 64x64 to exclusively accompany the first sprite. Then, in the Update() method, only half of the whole texture is rendered, properly displaying the sprite in use.
