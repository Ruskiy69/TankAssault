Touching at endpoints qualifies as intersections.

TRUE if they intersect, FALSE otherwise.
See also:
http://gamedev.stackexchange.com/questions/26004/how-to-detect-2d-line-on-line-collision

Solve for the intersection point.

Derivation: First line: y - y1 = m1(x - x1) Second line: y - y2 = m2(x - x2) Equal to each other: m1x - m1x1 + y1 = m2x - m2x2 + y2 Solve for 'x': m1x - m2x = -m2x2 + y2 + m1x1 + y1 x(m1 - m2) = -m2x2 + y2 + m1x1 + y1 x = (-m2x2 + y2 + m1x1 + y1) / (m1 - m2) Solve for 'y': y = m1(x) - m1x1 + y1 Thus (x, y) becomes the intersection point.

Then we must see if (x, y) is in the range of line segments given.
