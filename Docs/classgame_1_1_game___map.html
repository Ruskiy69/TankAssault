Detailed Description

The lowest layer is the terrain map. It describes terrain tiles, each which have unique properties such as impassability or slower movement speed. The next highest layer is the collision map. This one describes collision data for all on-screen tiles. Then finally, at the highest level, is the enemy path map, which specifies possible paths for enemies to follow.
