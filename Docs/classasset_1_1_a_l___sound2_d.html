bool AL_Sound2D::Play ( )

Due to the somewhat complicated source-management system in place, buffers are bound to an OpenAL source on demand. So whenever this method is called, an available source is found, and then the buffer is bound to it, and the audio is played. The AL_AssetManager is responsible for unloading the source after playback has completed in full.

Returns:
TRUE if the sound played, FALSE if not.
